# CRUD APP

aplikasi yang dibuat untuk pembelajaran saya dalam react js, express js, JWT, MySQL, Auth

## Tech Stack

**FrontEnd:** ReactJS (Vite), Axios, Bootstrap, React Router

**BackEnd:** Node, Express, JWT, Auth Login, MySQL

## Installation

**FrontEnd:**

```bash
  cd crud-frontend
  npm install
  npm run dev
```

**BackEnd:**

```bash
  cd crud-backend
  npm install
  node server.js
```
